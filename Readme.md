- 12-Variant (Mustaqil ish) ma'lumotlari bilan ishlash

- 5 xil bashorat sohasi:
  - Samarqand markazida uy narxlari
  - Kiberhujumlar zarari
  - Elektromobillar savdosi hajmi
  - Xalqaro aviachiptalar narxi

- Grafik va jadval ko'rinishida ma'lumotlar
- Kelajak bashoratini ko'rish

## 🚀 Ishga tushirish

### Lokal kompyuterda

1. Repository ni klon qilish:

```bash
git clone https://github.com/<username>/ai-bashorat.git
cd ai-bashorat
```

2.Virtual muhit yaratish va faollashtirish:

```bash
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Mac/Linux
```
3.Kerakli paketlarni o'rnatish:

```bash
pip install -r requirements.txt
```

4. Ilovani ishga tushirish:

```bash
streamlit run main.py
```
